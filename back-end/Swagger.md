## 테스트 환경 설정

서버가 http://localhost:8000에서 뜬다고 가정하면:

Swagger UI 화면:
http://localhost:8000/swagger-ui/index.html
OpenAPI 3 Library for spring-boot

OpenAPI JSON:
http://localhost:8000/v3/api-docs (기본 경로)
OpenAPI 3 Library for spring-boot

### CLI 명령어
```bash
// 빌드
./gradlew clean build
./gradlew clean build -x test
// 실행
./gradlew bootRun
```
