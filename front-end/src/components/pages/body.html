남한 지역 지도 기반 여행 시작

로그인 전 기본 화면에서 바로 지역을 선택할 수 있도록 남한 지도를 배치했습니다. 각 시·도는 파스텔 톤으로 구분했고, 서울을 1번으로 시작하는 검정색 순번을 표시했습니다. 번호는 인구 규모 중심으로 구성했고, 지도 영역 크기는 화면 가독성을 위해 지역 규모를 참고한 시각화 방식으로 배치했습니다.

※ 이 지도는 메인 페이지 UI용으로 제작한 SVG 기반 시각화입니다. 실제 행정 경계 GIS 데이터와 완전히 동일한 정밀 지도는 아니며, 로그인 전 랜딩 페이지에서 지역 선택 UX를 빠르게 구현하기 위한 구조입니다.

개발 확장 포인트

현재 HTML 단독 실행 가능 구조입니다. 이후 프론트 프로젝트에 붙일 때는 아래 방식으로 확장하면 됩니다.

A각 region 요소에 id 부여 후 클릭 이벤트 연결

D로그인 전/후 헤더 메뉴 조건부 렌더링
